# Customer Shopping Behavior Analysis 
# Overview
A data analytics project that analyzes customer purchasing behavior using SQL, Python, and Power BI to uncover insights on revenue trends, customer segments, and product performance.

# Features
- Customer behavior and purchase pattern analysis
- Revenue comparison across segments
- Discount impact evaluation
- Product performance insights
- Interactive Power BI dashboard
- Data cleaning and preprocessing in Python
- SQL based business queries for insights
# Technologies Used
Python(Pandas)
SQL
Power BI
Data Visualization
