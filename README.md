# Customer Shopping Behavior Analysis

# Overview
A data analytics project that analyzes customer purchasing behavior using Python, SQL, and Power BI. The project focuses on understanding revenue trends, customer segmentation, discount usage, and product performance using real dataset analysis.

# Features
- Loaded and explored customer shopping dataset using Python (Pandas)
- Data cleaning and preprocessing for analysis
- Created SQLite database and connected it with Python
- Performed SQL-based business queries for insights (Q1–Q10)
- Revenue analysis by gender, age, and subscription status
- Discount usage and spending behavior analysis
- Product performance and rating analysis
- Customer segmentation based on purchase history
- Identified top products and revenue contributors
- Data-driven insights for business decision making

# Technologies Used
- Python (Pandas)
- SQL 
- Jupyter Notebook
- Power BI (for dashboard visualization)

# Key Insights
- Revenue comparison across gender and customer groups
- Subscription status impact on spending behavior
- Discount users and their purchasing patterns
- Top performing products based on sales and ratings

# Project Workflow
- Imported dataset using Pandas
- Explored and validated data structure
- Created SQLite database from dataset
- Executed SQL queries for business analysis
- Derived insights from query results
- Prepared data for visualization in Power BI

# Dataset Columns
- Customer ID
- Age
- Gender
- Item Purchased
- Category
- Purchase Amount (USD)
- Location
- Review Rating
- Subscription Status
- Discount Applied
- Previous Purchases
- Shipping Type
